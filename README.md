# Getting Started
1. to entry Swagger , please add "/swagger" after http://localhost:9000
2. to entry Spring H2 Console , please add "/h2-console" after http://localhost:9000
   - JDBC URL  : jdbc:h2:file:./file/demoDB
   - User name : root
   - ![img.png](img.png)


# Build and Run

1. mvn clean install
2. mvn spring-boot:run
